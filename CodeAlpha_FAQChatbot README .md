# ⬡ NOVA — AI Knowledge Chatbot | CodeAlpha

A futuristic glassmorphism AI chatbot that answers general knowledge questions using NLP cosine similarity.

## ✨ Features
- 🤖 NLP-powered FAQ matching using cosine similarity
- 💬 Typewriter animation for bot responses
- 🌌 Animated starfield + glowing background
- 💡 Quick suggestion chips
- 📚 20+ knowledge topics (science, history, tech, AI, geography)
- ⚡ Zero external dependencies — pure Python!

## 🛠️ Run It
```bash
python main.py
```
No pip install needed!

## 🧠 How It Works
1. User types a question
2. App tokenizes input and computes cosine similarity against FAQ database
3. Returns the best matching answer with typewriter effect
